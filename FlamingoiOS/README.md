# Ocean iOS

SwiftUI starter for the iPhone version of Ocean / A G.Resto.

## Included screens
- Dashboard
- Reservations
- Workers
- Inventory
- Settings

## Notes
- This scaffold mirrors the Android and web data model.
- It is ready to be opened in Xcode on macOS.
- Firebase integration is intentionally isolated in `Services/FlamingoStore.swift`.

## Next step
Create an Xcode project and drop these source files into it, or use them as the basis for a new SwiftUI app target.
