## ✅ PROCHAINES ÉTAPES (RAPIDES)

### 1️⃣ CONFIGURATION FIREBASE (15 min)
□ Aller sur https://console.firebase.google.com
□ Créer projet "FlamingoAndroid"
□ Ajouter app Android: com.example.oceanandroid
□ Télécharger google-services.json
□ Placer dans app/
□ Activer Authentication (Email/Password)
□ Créer Firestore Database
□ Ajouter règles de sécurité

### 2️⃣ VÉRIFIER COMPILATION (5 min)
```powershell
cd "c:\Users\waelb\Desktop\projet ocean"
./gradlew clean build
```

### 3️⃣ INSTALLER ET TESTER (5 min)
```powershell
./gradlew installDebug
# OU depuis Android Studio: Run > Run 'app'
```

### 4️⃣ CRÉER DONNÉES DE TEST (10 min)
□ Ajouter utilisateur test dans Firebase Auth
□ Créer quelques documents dans Firestore
□ Tester CRUD operations
